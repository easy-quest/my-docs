Conditional statements in java‚Äã

Conditional statements are used to perform different actions based on different conditions. There are 2 ways of doing this in java

  1. If else
  2. Switch case

If else Code‚Äã

class Main {
  public static void main(String[] args) {
    int num = 5;
    if(num > 10){
      System.out.println("num is greater than 10");    
    } else if(num > 2){
      System.out.println("num is greater than 2");
    }    
  }
}

Switch case code‚Äã

class Main {
  public static void main(String[] args) {
    int num = 5;
    switch(num){
      case 1: System.out.println("Num is 1");
              break;
      case 3: System.out.println("Num is 3");
              break;
      case 5: System.out.println("Num is 5");
              break              
    }    
  }
}

Loops in java‚Äã

A loop allows to repeatedly execute a group of statements. There are 4 kinds of loops in java

  1. While loop
  2. Do while loop
  3. For loop
  4. foreach loop

While loop code‚Äã

class Main {
  public static void main(String[] args) {
    int[5] numbers = {1, 2, 3, 4, 5};
    int size = numbers.length;
    int i = 0;
    while(i < size){
      System.out.println(numbers[i]);
      i++;
    }
  }
}

Do while loop code‚Äã

class Main {
  public static void main(String[] args) {
    int[5] numbers = {1, 2, 3, 4, 5};
    int size = numbers.length;
    int i = 0;

    do{
      System.out.println(numbers[i]);
      i++;
    } while(i < size);
  }
}

Java Singleton‚Äã

In object-oriented programming, a singleton class is a class that can have only one object (an instance of the class) at a time.

Code‚Äã


public class SimpleSingleton {
    private static SimpleSingleton singleInstance =  new SimpleSingleton();

    //Marking default constructor private
    //to avoid direct instantiation.
    private SimpleSingleton() {
    }

    //Get instance for class SimpleSingleton
    public static SimpleSingleton getInstance() {

        return singleInstance;
    }
}
